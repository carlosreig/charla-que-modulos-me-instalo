Antes de empezar...

Vamos a ir rápido. Las diapositivas son solo para referencia. HAbrá algo de lío. Interrupciones.

Drush

Instalación

composer global require drush/drush:7.*
export PATH="$HOME/.composer/vendor/bin:$PATH" #añadir esta línea al final del archivo ~/.bashrc

Uso

drush en module_name

drush dl wysiwyg-7.x-2.x.http://ftp.drupal.org/files/projects/wysiwyg-7.x-2.x-dev.tar.gz. http://download.cksource.com/CKEditor/CKEditor/CKEditor%204.4.7/ckeditor_4.4.7_full.zip Se pueden configurar las opciones para cada text format
